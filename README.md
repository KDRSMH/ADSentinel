# ADWatchdog 🐾
> Active Directory Security Audit & Hardening Tool

ADWatchdog detects misconfigurations and security vulnerabilities
in Active Directory environments.

## Features (Roadmap)
- [ ] Kerberoastable account detection
- [ ] AS-REP Roasting vulnerability detection  
- [ ] Password policy analysis
- [ ] Privileged group audit
- [ ] HTML/JSON reporting
- [ ] Web dashboard
